Gemfile 必須先加上 gem ‘capistrano_colors’

cron 設定, 透過 whenever gem 在 deploy 時, 更新 cron
Gemfile 要有 gem ‘whenever’

multistages 設定, 讓 server 端的設定和佈署工作各自獨立管理, 同時資源佈署多個 server
